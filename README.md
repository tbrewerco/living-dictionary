# Living Dictionary

A simple dictionary web application that allows users to search for the definition of a word of their choice.

## Technologies
- HTML5
- CSS3
- JavaScript/jQuery
- [Merriam-Webster API](https://www.dictionaryapi.com/products/api-collegiate-dictionary)
